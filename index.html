BuaBook Open Source Projects

This website provides documentation for all the open-sourced projects provided by BuaBook.

Project list:

  • sf-et-fuel-sdk
    • Extension classes for use with interfacing with the ExactTarget API (building on the FuelSDK)
    • Documentation
  • http-common
    • HTTP Access and JSON Serialisation library
    • Documentation
  • spring-common
    • Spring auto-configuation library
    • Documentation
  • amazon-s3-file-transfer
    • Provides file upload and download wrapper classes for Amazon AWS SDK
    • Documentation
